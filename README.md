# Renamer
Rename file with names from another file

Work like Atomic Warez Leecher (AWL)

To use first input file extension and name your programm 
which you want rename. After, input file name with names (in repository already have one) and wait.
Programm sometimes create fantom files to delete use 7zip. Right click on all fantom files, in menu select "7zip", add to archive, delete all files after
 squeezing, and delete archive.
